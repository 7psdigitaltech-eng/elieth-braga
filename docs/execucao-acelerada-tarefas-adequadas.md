# Tarefas adequadas para execução acelerada

Exemplos adequados:

- implementação de página estática com conteúdo já aprovado;
- refinamento visual dentro de tokens já definidos;
- criação de componentes sem novas dependências;
- ajustes de responsividade;
- melhorias de acessibilidade previstas;
- organização de conteúdo já autorizado.

Exemplos inadequados:

- escolha de stack;
- deploy;
- CI/CD;
- integrações externas;
- imagem real;
- fatos novos;
- conteúdo eleitoral;
- qualquer decisão estratégica.
